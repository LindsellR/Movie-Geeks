JWTStrategy

JWT authentication strategy. Extracts JWT token from Authorization header and verifies the token. Finds user by ID embedded in the JWT payload.
Source:
  • passport.js, line 64

LocalStrategy

Local authentication strategy using username and password. Verifies user credentials by checking the username and password against the database. Calls `callback` with user info on success or false on failure.
Source:
